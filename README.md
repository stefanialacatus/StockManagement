# StockManagement

StockManagement is a web application that provides recommendations for optimizing the stock distribution across multiple store locations, helping businesses maximize profit while minimizing transportation costs.

### Features:
- Authentication as admin or store manager
- Admin: Inventory monitoring across all store locations
- Store Manager: Inventory monitoring for the alocated store
- Stock Status alerts (Understock, OK, At Risk, Overstock, Dead Stock)
- Automatic stock transfer recommendations

### Tech Stack:
- Frontend: React, CSS
- Backend: Python, Flask, OR-Tools
- Database: PostgreSQL
